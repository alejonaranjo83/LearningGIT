Blogpost para aprender GIT

GIT es un sistema de control de versiones (VCS) permite guardar el historial de cambios y crecimiento de los archivos de un proyecto. Aplica para textos planos.

Realiza actualizaciones solamente en los lugares donde hay cambios, en vez de guardar un mismo archivo varias veces.

Lleva un registro de los cambios que otras personas realicen en los archivos + permite reducir errores humanos + detectar fallos de forma más rápida y eficiente.
